# Kari Markus | Portfolio

Personal portfolio website showcasing my skills, projects, and background as a career-changing software developer. Built with vanilla technologies and a minimalist, modern design approach.

## Live Demo

https://markar07.github.io/Portfolio/

## Technologies

- HTML5, semantic markup
- CSS3, custom properties and CSS Grid
- Vanilla JavaScript, no frameworks
- Formspree, contact form handling
- GitHub REST API, real-time profile statistics
- GitHub Pages, hosting

## Design

Modernism and minimalism are the guiding themes. Cormorant Garamond serif headings create a contrast with Inter body text. The color palette builds on muted dark or light backgrounds with an orange accent color.

The background features a subtle dot pattern that fades at the edges with a vignette effect and stays in place as the user scrolls (parallax). Together these create depth without distracting from the content.

## Features

- Bilingual content, Finnish and English, switchable on the fly
- Dark and light theme toggle, preference stored in localStorage
- Left-side section navigation that follows the active section while scrolling
- Flip cards for skills, revealing details on the back
- Masonry-style project grid with varying card sizes
- Project modals with image gallery and keyboard navigation
- Real-time GitHub statistics via REST API integration
- Contact form with Formspree integration
- Responsive from mobile to desktop

## Accessibility

- Semantic HTML structure
- Aria-label attributes for interactive elements
- Visually hidden text for screen readers
- Keyboard navigation for flip cards and modals
